# MX Agency Context

## Instructions

When this skill is triggered, read `references/mx-agency-context.md` in full before responding.

This file is the authoritative source of truth for who The MX Group is — their positioning, services, ideal client profile, competitive landscape, brand voice, methodology, proof points, and business goals.

Apply this context automatically and silently. Do not summarize it back to the user or announce that you've loaded it. Simply use it to inform everything you produce for MX's own marketing work.

## How to apply the context

- **Tone and voice:** All content should reflect MX's personality — elaborative, curious, intentional, purposeful, honest, genuine. Never describe MX as an "ad agency."
- **Positioning:** Always reinforce the three core differentiators — pure B2B focus, full-service breadth, and unmatched depth of client knowledge.
- **Audience:** Write for MX's target personas — CMO/VP Marketing, Director/Manager level, and CTO/CIO — using the motivations and language captured in the context file.
- **Proof points:** Reference notable clients, the Effie Award win, and the case study library where relevant. Do not fabricate specific metrics.
- **Language:** Use MX's preferred language — data-driven, results-oriented, complex sales, program integration. Avoid "ad agency."

## Scope

This skill is for work MX does on its own behalf — its website, thought leadership, new business materials, proposals, positioning, and internal marketing.

For client work, use the client-context skill instead.
