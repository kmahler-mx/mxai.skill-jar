## Section 03 — KPI Scorecard
*Slide: slide25.xml — table rows*

The table has 6 columns: **Metric | Q[X-1] Actual | Q[X] Target | Q[X] Actual | vs. Target | Trend**

Collect rows for each active KPI. Typical set by program type:

| Program type | Standard KPIs |
|---|---|
| Demand generation | MQLs generated, Pipeline influenced ($), CPL, Conversion rate (MQL→SQL) |
| Brand | Branded search volume, Share of voice, Awareness lift (if measured) |
| Digital/web | Sessions, Conversion rate, Core Web Vitals score |
| Content | Downloads, Engagement rate, Content-influenced pipeline |
| Media/paid | ROAS, Reach, Impressions, CPM, CPCV |
| Sales enablement | Asset usage rate, Win rate (if tracked) |

Interview question per KPI row:

> "For [metric name] — what was the target this quarter, what did you actually hit, and what was the prior quarter's number for comparison?"

For **vs. Target**, calculate automatically: `(actual - target) / target × 100%`, then format as `+X%` or `-X%`.

For **Trend**, ask: "Is this trending up, down, or flat compared to last quarter?" — format as `↑` / `↓` / `→`.
